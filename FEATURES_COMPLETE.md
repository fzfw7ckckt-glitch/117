## 🚀 ШВИДКИЙ СТАРТ

### Запуск
```bash
cd osint-platform-2026
docker-compose up -d
```

### Доступ
- Frontend: http://localhost:3000
- API: http://localhost:8000
- API Docs: http://localhost:8000/docs

### Генерація звіту
```bash
# Резюме
curl http://localhost:8000/reports/{id}/summary

# Аналіз загроз
curl http://localhost:8000/reports/{id}/analysis

# Статистика
curl http://localhost:8000/reports/{id}/statistics

# Повний звіт (JSON)
curl -X POST http://localhost:8000/reports/{id}/generate-report?format=json

# Експорт (HTML)
curl -X POST http://localhost:8000/reports/{id}/generate-report?format=html > report.html

# Експорт (Markdown)
curl -X POST http://localhost:8000/reports/{id}/generate-report?format=markdown > report.md

# Експорт (CSV)
curl -X POST http://localhost:8000/reports/{id}/generate-report?format=csv > report.csv
```
